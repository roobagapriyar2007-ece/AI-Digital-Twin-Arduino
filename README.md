# AI Digital Twin Based EHV 400/220 kV Substation Monitoring System

## Overview
This project presents an AI-enabled Digital Twin for an EHV 400/220 kV Substation. It provides real-time monitoring of important substation parameters such as voltage, current, temperature, humidity, oil level, vibration, and frequency. The system helps detect abnormal conditions and improves substation safety and reliability.

## Features
- Real-time sensor monitoring
- AI-based fault prediction
- Digital Twin dashboard
- Transformer monitoring
- Circuit breaker monitoring
- Protection relay monitoring
- Live alarms and notifications
- Cloud data monitoring using Firebase
